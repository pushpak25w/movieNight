# Movies Night
## use:
### for [server](http://127.0.0.1:5000/):
    cd nodejs-chat
    cd server
    npm i
    npm start
    
### for [client](http://127.0.0.1:3000/):
    cd nodejs-chat
    cd client
    npm i
    npm start
### recommendation [api](http://127.0.0.1:6000/category/): 
    pip install -r requirements.txt 
    ./bootstrap
