# Movies Night

## use:
### for server runs on 127.0.0.1:5000:
    cd nodejs-chat
    cd server
    npm i
    npm start
    
### for client runs on 127.0.0.1:3000:
    cd nodejs-chat
    cd client
    npm i
    npm start
### recommendation runs on 127.0.0.1:6000: 
    pip install -r requirements.txt 
    ./bootstrap
```use 127.0.0.1 not localhost```
